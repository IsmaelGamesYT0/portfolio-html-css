# Portfólio HTML & CSS

Este repositório contém meu portfólio de desenvolvimento web utilizando **HTML5** e **CSS3**.  
O objetivo é reunir páginas demonstrativas, desafios e práticas voltadas ao aprendizado e à apresentação de conceitos fundamentais da web.

---

## Tecnologias utilizadas
- HTML5
- CSS3

---

## Estrutura do projeto

Portfolio-HTML-CSS/
├── index.html # Página principal (GitHub Pages)
├── style.css # Estilos globais
├── demos/ # Demos e páginas de estudo
│ ├── desafios/
│ ├── exercicios/
│ └── praticas/
├── README.md
├── CONTRIBUTING.md
└── LICENSE.md


- A raiz do projeto é usada como **GitHub Pages**.
- Cada subpasta dentro de `demos/` contém um `index.html` acessível diretamente pelo navegador.

---

## Como usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/IsmaelGamesYT0/Portfolio-HTML-CSS.git

2. Abra o arquivo index.html no navegador
ou acesse via GitHub Pages, se disponível.

3. Explore as pastas em demos/ para ver os desafios, exercícios e práticas.
